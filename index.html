.container {
  display: flex;
  flex-direction: row;
  flex-wrap: nowrap;
  justify-content: flex-start;
  align-items: stretch;
  align-content: stretch;
  gap: 0;
}
flex-direction: row - Items are placed in a row, from left to right.

Flexbox Item Properties

Modify properties for flex item #3 to see how they affect an individual item:

.item:nth-child(3) {
  flex-grow: 0;
  flex-shrink: 1;
  flex-basis: auto;
  align-self: auto;
  order: 0;
}
flex-grow: 0 - The item will not grow to fill available space.
